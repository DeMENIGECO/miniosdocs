Iniziare con miniOS
===================

Qui potrai iniziare con miniOS. Installerai e configurerai il tuo miniOS.

Scaricalo
---------

Vai su `demenigeco.github.io/miniOS/ <https://demenigeco.github.io/miniOS/>`_, clicca "Scarica" e poi scarica l'ultima versione.

Avvialo e fai l'OOBE
--------------------
Il sito ha scaricato una zip, estraila e apri il file.

.. warning::

   È necessario estrarre il file ZIP per il corretto funzionamento.

Poi dovrebbe apparire un'orologio. Cliccaci e poi imposta quanto:

- Nome PC
- Nome utente
- Accento
- Tema chiaro/scuro

.. tip::
  Se desideri avere le trasparenze (come Aero), allora imposta il tema scuro. Se desideri colori più piatti senza trasparenze, usa il tema chiaro.

Fine
----
Ora hai finito.
